## How to Play

1. Open `index.html` in your web browser
2. Player X goes first
3. Click on any empty cell to place your mark
4. Players alternate turns
5. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins!
6. If all cells are filled without a winner, it's a draw
7. Click the "Restart Game" button to start a new game

## Game Rules

- The game is played on a 3x3 grid
- Two players take turns marking cells
- Player X always starts first
- Players cannot mark an already occupied cell
- The game ends when:
  - A player gets three marks in a row (win)
  - All cells are filled without a winner (draw)
